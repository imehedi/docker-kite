[![Build Status](https://github.com/imehedi/docker-kite/workflows/Docker-Builder/badge.svg)](https://github.com/imehedi/docker-kite/actions)

## What is kite-docker?

This is a docker image with essentials for orchestrating Kubenetes on AWS. There may as well be other such images, please free to use this or the others.

## How to use?

You can do a docker pull with the following 
```docker pull imehedi/k8s-manager```

Alternatively you can clone the repo and build with your own AWS credentials (aws/credentials) and the attached Dockerfile.

No logical planning, just the name that came to mind
